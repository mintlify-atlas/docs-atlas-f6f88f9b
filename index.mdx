## Welcome to intercom-ruby

`intercom-ruby` gives you a simple, idiomatic Ruby interface to the full Intercom REST API. Create contacts, manage conversations, track events, and send messages — all from your Ruby or Rails app.

## Get started in 3 steps

<Steps>
  <Step title="Install the gem">
    Add `intercom` to your Gemfile or install it directly:

    ```bash
    gem install intercom
    ```
  </Step>
  <Step title="Initialize your client">
    Create a client using your Personal Access Token:

    ```ruby
    require 'intercom'

    intercom = Intercom::Client.new(token: ENV['INTERCOM_ACCESS_TOKEN'])
    ```
  </Step>
  <Step title="Make your first API call">
    Create a contact or list your existing ones:

    ```ruby
    # Create a new contact
    contact = intercom.contacts.create(email: 'user@example.com', role: 'user')

    # List all contacts
    intercom.contacts.all.each { |c| puts c.email }
    ```
  </Step>
</Steps>
